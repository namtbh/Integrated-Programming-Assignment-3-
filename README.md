    <h1>README for BST, AVL Tree, and Max Heap Implementation</h1>

    <h2>Overview</h2>
    <p>
        This project contains two Python scripts:
    </p>
    <ul>
        <li><strong><code>file1.py</code></strong>: Implements a Binary Search Tree (BST) and an AVL Tree to add and search nodes, comparing their performance with a Python list.</li>
        <li><strong><code>file2.py</code></strong>: Implements a Max Heap that organizes employees by salary, displaying them in descending order.</li>
    </ul>

    <h2>Files</h2>
    <ul>
        <li><strong><code>file1.py</code></strong>: Compares search operations between a list, BST, and AVL Tree.</li>
        <li><strong><code>file2.py</code></strong>: Uses a Max Heap to sort employee data by salary.</li>
    </ul>

    <h2>How to Run</h2>
    <p>To execute:</p>
    <pre><code>python file1.py</code></pre>
    <pre><code>python file2.py</code></pre>

    <ul>
        <li><strong><code>file1.py</code></strong>: Inserts 150,000 random numbers into a list, BST, and AVL Tree, then searches and compares performance.</li>
        <li><strong><code>file2.py</code></strong>: Inserts various employees into a max heap and displays them by descending salary.</li>
    </ul>

    <h2>Requirements</h2>
    <ul>
        <li>Python 3.7+</li>
